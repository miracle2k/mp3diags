You may want to look at change log for your program here, especially if you installed an older version before.
While MP3 Diags is quite usable, it is currently in active development and new versions are released once or twice a week. Usually they contain rather minor tweaks, or perhaps changes that would only affect some users, but a more serious bug fix may also be included. I generally don't like the idea of always downloading the latest version just because it's the latest; I'd rather keep using something that I already tested and it seemed to work fine. However, from time to time you may want to check the MP3 Diags blog, where announcements about new releases are made. Then you can decide based on what changed in a given release if you want to upgrade or not. (If you use an RSS reader, you can add that page to it, so you see the notifications automatically.)
You can also look at change log at the project's site, to see what changed since you installed your version
There are both source files and binaries, for several Linux platforms, as well as for 32bit Windows. More details can be found at the main MP3 Diags download page.
